	function deepClone(obj) {
		let newObj = obj instanceof Array ? []:{};
		if(typeof obj !== 'object') {
			return obj;
		}else{
			for(var i in obj) {
				newObj[i] = typeof obj[i] === 'object' ? deepClone(obj[i]) : obj[i];
			}
		}
		return newObj;

（1）定位方案
（1.1）普通流 (normal flow)
行内元素水平排列，块级元素换行排列。默认为普通流定位方案。
（1.2）浮动 (float)
在普通流基础上，增加浮动
（1.3）绝对定位 (absolute positioning)
脱离普通流

（2）
BFC即Block Formatting Contexts(块级格式化上下文)，属于定位方案的普通流。

（3）Box
（3.1）Box是css布局的基本单位
（3.2）Box类型
block-level box:display属性为block,list-item,table。参与BFC即Block Formatting Contexts
inline-level box:display属性为inline,inline-block,inline-table。参与IFC即Inline Formatting Context

（4）BFC布局规则
（4.1）块盒与行盒（行盒由一行中所有内联元素组成）margin的左边，与父元素border 的左边相接触(对于从左往右的格式化，否则相反)。即使存在浮动也如此
（4.2）块盒与行盒（行盒由一行中所有内联元素组成）沿垂直方向，一个接一个排列
（4.3）块盒与行盒（行盒由一行中所有内联元素组成）垂直方向距离由margin决定。属于同一BFC两相邻盒的margin会重叠
（4.4）BFC区域不与float盒重叠
（4.5）BFC是独立布局环境，其中元素布局不受外界影响，也不影响外界
（4.6）计算BFC高度时，浮动元素也参与计算。

（5）创建BFC
（5.1）float值不是none。
（5.2）position值不是static或relative。
（5.3）display值是inline-block、table-caption、table-cell、flex或inline-flex
（5.4）overflow值不是visible
